Rubyによるスクレイピングが学べるウェブサイト

Jan 8th, 2014 9:39 am | Comments

Rubyを使ったウェブのスパイダリング、スクレイピングを学べる情報源がないか探していたところこちらのサイトを見つけました。

  • The Bastards Book of Ruby

トップページから「Contents」「Supplements」と順に追っていくと、ウェブ・スクレイピングの基礎をテーマにした以下の記事が見つかります。

  • An Intro to Web Scraping
  • Meet Your Web Inspector
  • Inspecting a Webpage’s Traffic
  • Parsing HTML with Nokogiri
  • Writing a Web Crawler
  • The Mechanize Gem

スクレイピングの基本的な考え方から始まって、ブラウザ付属のインスペクタの使い方、「Mechanize」の紹介と、ひととおりの情報がまとまっています。

実用的なスクリプトを書くにはまだ少しハードルがありそうですが、基本的な考え方はこちらの記事で理解できたと思います。復習を兼ねてサンプルスクリプトを少しいじってみました。

1. Wikipedia 検索ボックスのサジェスト情報を取得する

「Inspecting a Webpage’s Traffic」のサンプルを元に、Wikipediaの検索ボックスからサジェスト情報を取得してみます。元のサンプルでは英語版のWikipediaを対象にアルファベット毎のサジェスト情報を取得しています。少し手を加えて日本語版のWikipediaから五十音毎のサジェスト情報を取得してみます。

できあがったのものはこちら。

最初にrequireしている「crack」は、調べてみるとJSON/XML用のシンプルなパーサでした。

リクエストの送信に「open-uri」でopenするのでも問題はないと思われます。

ひらがなはそのままでは送信することができないので、CGI.escapeでURIエンコードしています。もしかすると、URI.encodeを使う方が正しいのかも。使い分けがあまり理解できていません。

実行すると、Wikipediaの検索ボックスにひらがな一文字を入力したときサジェストされる単語の一覧が取得できます。
